# Настройка DNS записей

Для работы системы необходимо настроить DNS записи для домена `skyrodev.ru`.

## A записи (IPv4)

Создайте следующие A записи, указывающие на IP `95.31.44.165`:

```
api.skyrodev.ru          A    95.31.44.165
admin.skyrodev.ru        A    95.31.44.165
grafana.skyrodev.ru      A    95.31.44.165
prometheus.skyrodev.ru   A    95.31.44.165
minio.skyrodev.ru        A    95.31.44.165
traefik.skyrodev.ru      A    95.31.44.165
```

## Wildcard запись (опционально)

Для автоматической поддержки всех поддоменов можно создать wildcard запись:

```
*.skyrodev.ru            A    95.31.44.165
```

## Проверка DNS

После настройки DNS проверьте разрешение имен:

```bash
nslookup api.skyrodev.ru
nslookup admin.skyrodev.ru
```

Все записи должны указывать на `95.31.44.165`.

## Примечание

DNS изменения могут занять до 48 часов для полного распространения (TTL обычно 3600 секунд).
